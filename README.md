# 💰 Expense Tracker

A modern **Personal Finance Management Web App** built with **Flask + MongoDB** to manage expenses, income, savings, borrow/lent records, and generate reports.

---

## 🚀 Features

### 🔐 Authentication
- User Signup & Login
- Secure Password Hashing
- Confirm Password Validation
- Unique Email Validation
- Invalid Login Popup
- Duplicate Email Popup
- Remember Me Feature
- Show/Hide Password

---

### 💸 Expense Management
- Add Expenses
- Edit Expenses
- Delete Expenses
- Category-wise Tracking
- Payment Mode Tracking
- Expense Analytics

---

### 💵 Income Management
- Add Income
- Income Source Tracking
- Payment Mode Tracking
- Monthly Income Analytics

---

### 📊 Dashboard
- Total Expense Overview
- Total Income Overview
- Remaining Budget
- Monthly Financial History
- Carry Forward Logic
- Category-wise Charts
- Monthly Expense Charts
- Income Charts

---

### 🤝 Borrow & Lent System
- Add Borrow Records
- Mark Borrow as Paid
- Add Lent Records
- Mark Lent as Received
- Track Pending/Cleared Status

---

### 🎯 Savings Goals
- Create Savings Goals
- Update Savings Progress
- Progress Bar Tracking
- Remaining Savings Calculation

---

### 📑 Reports
- Export PDF Reports
- Export CSV Reports
- Financial Summary Reports

---

### 🎨 UI/UX
- Dark Theme
- Light Theme
- Theme Toggle
- Animated Error Popups
- Responsive Dashboard
- Modern Finance UI
